VenforceGo — Extensão

Baixe a extensão usada para apoiar rotinas operacionais no Mercado Livre.

Após o download, mantenha a pasta descompactada em um local fixo para o Chrome continuar carregando a extensão corretamente.

Baixar extensão (.zip)

Como instalar

Instalação manual no Google Chrome.

  1. 1Baixe o arquivo .zip clicando no botão acima.
  2. 2Descompacte a pasta em um local fixo no seu computador. Não mova a pasta depois de instalar.
  3. 3Abra o Google Chrome e acesse chrome://extensions na barra de endereço.
  4. 4Ative o "Modo do desenvolvedor" no canto superior direito da página.
  5. 5Clique em "Carregar sem compactação" e selecione a pasta descompactada.
  6. 6A extensão VenforceGo aparecerá na lista. Pronto!

Como usar

Preparação local após baixar o arquivo personalizado.

  1. 1Baixe o arquivo .zip clicando no botão acima.
  2. 2Extraia a pasta no seu computador.
  3. 3Instale o Python caso ainda não tenha (python.org).
  4. 4No terminal, dentro da pasta extraída, instale as dependências:
    pip install playwright
  5. 5Em seguida rode:
    playwright install
  6. 6Execute o script principal:
    python v1_10_1.py
  7. 7Na primeira execução, faça login no Mercado Livre quando a janela abrir.
  8. 8A sessão será salva automaticamente para os próximos usos.
